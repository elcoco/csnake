# snek bot

A snake game AI written in C using ncurses.  
Can also be played manually but why would you do that?  
This project is very much in development.  

<img src="snek.jpg" width="200" />

## Controls

    h, ←    move left
    j, ↓    move down
    k, ↑    move up
    l, →    move right

    <space> toggle pause
    q       quit
